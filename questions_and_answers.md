###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A

<i>We know **let** is a variable keyword of javascript. Here is firstly declared a variable with **let**, then second line assigned an empty object value in greeting variable. We can use **let** when need to reassign a variable. Thats why when console the greeting, result will be empty object (because {} is the value of greeting variable). </i>

</p>
</details>
